Constants

There can be cases when you need to tell the program that the variable can't change its value throughout the program. Here come constants! Constants are similar to variables and are declared using the const keyword:

Constants must have a value when declared and they cannot change their value.

Rules of Naming Variables

Let's talk about names. There are several naming rules that you need to follow in JavaScript:

-variable names must begin with a letter, an underscore _ or a dollar sign $

-variable names cannot contain spaces

-variable names can only contain letters, numbers, underscores, or dollar signs.

-variable names are case-sensitive, which means that, for example, Name and name variables are different
